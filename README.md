# vue + springboot + 카카오 로그인 프레임

Vue3 + Spring Boot 기반의  
**카카오 소셜 로그인 기본 구성이 완료된 템플릿 프로젝트**입니다.  
사이드 프로젝트나 신규 서비스 시작 시 바로 복사하여 사용할 수 있도록 구성되어 있습니다.

## 📦 기술 스택

- Frontend: Vue 3 (Composition API)
- Backend: Spring Boot
- Auth: Kakao OAuth2
- DB: PostgreSQL (기본 설정 기준)

## ✅ 구현된 기능

- 카카오 로그인 (인가 코드 → 백엔드 전달 구조)
- 프론트 ↔ 백엔드 연동 기본 구조
- 사용자 정보 저장용 users 테이블 구조 ( 로컬db, db는 직접 만드세요 )
- 프로젝트 초기 세팅 프레임

1. 카카오 API 키 설정
- Frontend
- App.vue 또는 환경 변수에 Kakao JavaScript Key 설정

- Backend
- application.yml에서 Kakao rest api 키 설정

2. db 정보 변경

3. 프로젝트명 변경

- 패키지명
- application name
- 프론트 프로젝트명

각 프로젝트 목적에 맞게 변경 후 사용하세요.


⚠️ 주의사항

본 프로젝트는 프레임(템플릿) 용도입니다.

**운영 배포 시:**

- **JWT / 세션 처리 => JWT를 프론트에서 받기만하고 저장안함 => 쿠키 저장 권장**
- **카카오 Redirect URI => 운영 배포시 도메인에 맞는 url**
- **CORS 설정 => 운영에서 필수**
- **토큰 만료 후 처리 => 프론트 처리**

를 반드시 추가하세요.


## 🔐 인증(JWT)

- 카카오 로그인 성공 시 백엔드에서 JWT를 발급합니다.
- 발급된 JWT는 프론트엔드에서 저장 후 API 요청 시 사용합니다.
- 현재 프레임은 로그인 및 토큰 발급까지 구현되어 있으며,
  토큰 저장 방식(localStorage / cookie) 및 갱신 정책은
  프로젝트 성격에 맞게 확장하여 사용하세요.

  ## 🔁 카카오 로그인 흐름

1. 프론트에서 카카오 로그인 요청
2. 카카오 인가 코드 발급
3. Redirect URI로 이동
4. 백엔드에 인가 코드 전달
5. 사용자 확인 후 JWT 발급
6. 프론트에서 토큰 저장 및 메인 화면 이동
